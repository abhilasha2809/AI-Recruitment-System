## 📌 Overview
An intelligent recruitment system that automates the hiring process using **Artificial Intelligence and Machine Learning**.  
It helps recruiters filter candidates efficiently by analyzing resumes and matching them with job requirements.

---

## 🎯 Key Features

✨ Resume Screening using AI  
✨ Candidate Ranking System  
✨ Job Matching Algorithm  
✨ User-friendly Interface  
✨ Secure Data Handling  
✨ Real-time Processing
